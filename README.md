- `/GPTAPI` ==**已停止维护!!!**==

  是运行在北美服务器端的 ASP.NET Core WebAPI 项目。==**已停止服务!!!**==

  是运行在北美服务器端的 Django WebAPI 项目。

  使用 Python 为后端语言, 运行在 Ubuntu Server 20.04 LTS 64位才做系统.
